Intro to Markdown

2019-02-03
Markdown

最近發現用Markdown來做筆記超好用的！
如果是平常用pages、word或記事本，常會因為寫完沒好好整理導致忘了什麼東西寫在哪，又或是整理起來很花時間，到最後又懶得整理了XDD

Markdown常見於Github的readme.md檔
他的語法不會像HTML那樣繁雜，相反的他可以用很簡單很簡單的語法就能表現各個樣式的文字～

- [x] @mentions, #refs, [links](), **formatting**, and <del>tags</del> supported

- [x] this is a complete item
- [ ] this is an incomplete item
