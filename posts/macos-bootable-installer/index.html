製作 macOS 開機隨身碟

2020-05-21
Mac

製作系統開機隨身碟之前要先到App Store下載Mac OS X作業系統
但有時候舊版本的OS已經找不到了該怎麼辦？
以下是我蒐集的一些下載連結…

下載舊的 macOS 檔案 

OS X 10.10 Yosemite (dmg)
OS X 10.11 El Capitan (dmg)
macOS 10.12 Sierra (dmg)
macOS 10.14 Mojave

製作開機隨身碟 

  1. 插入要做為可開機安裝磁碟的 USB 隨身碟或其他卷宗

  2. 確認macOS Installer已經存在於/Applications

  3. 確認USB的名稱 (這邊以MyVolume代替)
    可以使用ls /Volume指令查看名稱

  4. 在終端機輸入指令後輸入你的密碼 (輸入密碼時，「終端機」不會顯示任何字元)

    El Capitan

    sudo /Applications/Install\ OS\ X\ El\ Capitan.app/Contents/Resources/createinstallmedia --volume /Volumes/MyVolume --applicationpath /Applications/Install\ OS\ X\ El\ Capitan.app --nointeraction

    Sierra

    sudo /Applications/Install\ macOS\ Sierra.app/Contents/Resources/createinstallmedia --volume /Volumes/MyVolume --applicationpath /Applications/Install\ macOS\ Seirra.app --nointeraction

    Mojave

    sudo /Applications/Install\ macOS\ Mojave.app/Contents/Resources/createinstallmedia --volume /Volumes/MyVolume

  5. 出現提示時，輸入Y確認您要清除卷宗，然後按下 Return 鍵。清除卷宗時，「終端機」會顯示進度
