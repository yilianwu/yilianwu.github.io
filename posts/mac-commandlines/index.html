一些常用到的指令筆記

Command Line Basics 一些基本ㄉ操作 

~ –>最上層根目錄，通常是/Users/username

ls -> 可看所有的內容

cd空格 資料夾名稱 –>進入某資料夾

cd空格 .. –>直接回到最上層根目錄

mkdir空格 資料夾 –>新增資料夾

touch空格 filename.txt –>新增一個filename.txt檔案

rm空格 filename.txt –>刪除檔案

rm空格 -r空格 dirname/ –>刪除一整個資料夾

command+上方向鍵 –>切換最近輸入的指令

mv空格 fileone.txt filetwo.jpg dirname/ –> 移動fileone.txt及filetwo.jpg 到dirname資料夾

mv空格 yo.html hey.html –>重新命名yo.html成hey.html

cp空格 hello.html test/ –> 複製hello.html到test資料夾

建立加密的 Zip 壓縮檔 

讓機密資料加上密碼保護!

zip -e archivename.zip filetoprotect.txt

壓縮資料夾

zip -er ~/Desktop/encrypted.zip ~/Documents/Confidential/

在 Mac 上重置 NVRAM 或 PRAM 

  1. 在 Mac 上，選擇「蘋果」選單 >「關機⋯」。

  2. 在鍵盤上找到下列按鍵：Option、Command、P 和 R。您需要在步驟 4 中同時按住這些按鍵。

  3. 將 Mac 開機。

  4. 在帶有蘋果標誌的黑色螢幕出現前，立即按住 Option + Command + P + R 鍵。

  5. 繼續按著按鍵，大約 15 秒，直到 Mac 重新開機。

  6. 放開按鍵。

重置 Mac 的 SMC 

重置系統管理控制器（SMC）可解決某些與電源、電池、風扇和其他功能相關的問題

  1. 在關機狀態，按住下列所有按鍵： 位於鍵盤左側的 Shift
    位於鍵盤左側的 Control
    位於鍵盤左側的 Option（Alt）
  2. 同時按住這四個按鈕不放 10 秒。
  3. 放開所有按鍵，然後按住電源按鈕來啟動 Mac。

關於 SMC 

「系統管理控制器」負責管理與下列功能相關的行為：

電源，包括電源按鈕與 USB 連接埠供電 電池與充電 風扇與其他散熱管理功能 指示器或感應器，如狀態指示燈（睡眠狀態、電池充電狀態等）、瞬間防震感測裝置、環境光度感測器，以及鍵盤背光 筆記型電腦上蓋開啟與關閉時的行為 重置 SMC 不會影響 NVRAM 或 PRAM 的內容。

Finder顯示隱藏檔 

  1. 在Finder按cmd+shift+<
  2. 終端機輸入 
    defaults write com.apple.finder AppleShowAllFiles -bool true

重啟Finder

killall Finder
