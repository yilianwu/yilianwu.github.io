AWS 

https://aws.amazon.com/

註冊一個帳號 

登入Console 

服務>運算>EC2
選擇EC2

Launch Instance 

創立一個Instance

Step 1: 選擇AMI 

請選擇Ubuntu Server 18.04 LTS (HVM), SSD Volume Type - ami-07ad4b1c3af1ea214

Step 2: Choose an Instance Type 

選擇免費方案！

Step 3: Review and Launch 

準備發佈，不用害怕

Step 4: Launch再給他按下去 

檢查無誤就可以發布囉

Step 5: 創建Key Pair 


創建好按Download
.pem檔會存在你的下載資料夾

終端機打開 

一些設定 

  1. cd ~/Downloads/
  2. chmod 400 檔名.pem
    將權限打開
  3. ls -al 檔名.pem
    若出現-r就代表成功了哦
  4. cat 檔名.pem


登入你的系統 

ssh -i "檔名.pem" ec2-13-114-52-101.ap-northeast-1.compute.amazonaws.com
ec2-13-114-52-101.ap-northeast-1.compute.amazonaws.com是我的Public DNS (IPv4)

可以從這裡看～

若出現綠色ubuntu@ip-xxx-xx-xx-xx
就代表成功登入ㄌ！
